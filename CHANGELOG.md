# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-03

### Added
- Initial release of **Developer-CLI-Tool** CLI.
- `init` command for project scaffolding.
- Vite + React + TypeScript support.
- TailwindCSS 3.x integration with custom brand colors.
- React Router 6.x data router implementation.
- Authentication context with JWT pattern.
- Axios interceptors and services setup.
- Professional modular directory structure.
- Inquirer-based interactive mode.
- Non-interactive mode support with CLI flags.
- Ora/CLI Spinners for status feedback.
- Chalk-based semantic coloring.
- ESLint + Prettier opinionated configuration.
- Absolute import support (`@/*`).
